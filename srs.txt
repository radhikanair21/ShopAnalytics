📄  Software Requirements Specification (SRS)
ShopAnalytics — Real-Time Admin Sales Dashboard
Version: 1.0 | Date: April 2026 | Team Size: 5 Members
1. Project Overview
ShopAnalytics is a web-based admin dashboard for a small online shop. It provides real-time
analytics including order tracking, revenue statistics, top-selling products, and category-wise
sales charts. The system uses Firebase as the backend database and authentication provider,
Redux Toolkit for global state management, and React (Vite) on the frontend.
2. Functional Requirements
ID Requirement Priority
FR-01Admin can log in using email and passwordHigh
FR-02Dashboard shows total orders count in real-timeHigh
FR-03Dashboard shows total revenue in real-timeHigh
FR-04Monthly revenue displayed as a line/area chartHigh
FR-05Category-wise sales displayed as a pie/donut chartHigh
FR-06Top 5 selling products shown in a ranked tableHigh
FR-07Full orders list with search and paginationHigh
FR-08Products list with sortable columns Medium
FR-09Logout clears session and redirects to loginHigh
FR-10Non-logged-in users are redirected to loginHigh
FR-11 Data updates automatically without page refreshHigh
